# Rolevio Web App

AI Brand Agent Platform - 让每个品牌都拥有自己的AI营销团队

## Features

- **Brand Agent** - 品牌守护者，学习并保持品牌一致性
- **Copywriter Agent** - AI文案生成
- **Designer Agent** - AI设计师
- **Strategy Agent** - AI策略师
- **Campaign Agent** - 多渠道投放
- **Analytics Agent** - 数据分析

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
